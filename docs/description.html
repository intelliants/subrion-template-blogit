Simple and stylish template for bloggers.

For best experience, set Blog as your homepage and modify main menu items accordingly.

To edit footer text line go to Admin Panel -> System -> Languages and hit Edit phrases. Find phrases by keys 'footer_text' and 'copyright'

Background image

Since version 3.3 of Blogit template, you can add your own header background. Go to Admin Panel -> Extensions -> Template Settings (little cog icon).

Comments

This template works great with mgDisqus plugin that adds Disqus comments system to blog entries.

After installing this plugin, do the following:

  • Register on Disqus.com
  • Add your short name here: Admin Dashboard / System / Miscellaneous

  • important Change Comments block position to 'comments': Admin Dashboard / Content / Blocks. Edit Comments block.
